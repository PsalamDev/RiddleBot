# RiddleBot

The RiddleBot Generator is a web application that allows users to select a category such as School, Food, Sports, or Tech and instantly receive an AI-generated joke and riddle. Users can interact with the content by rating whether they found it funny or not. The system uses AI to generate unique content and stores user feedback to measure engagement and improve future recommendations.
